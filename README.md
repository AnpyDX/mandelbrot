# Mandelbrot-Set image generater
This is a **Mandelbrot Set** image generater written in Rust.

<img src="./screenshot.png" align="right"/>

### Features
- multi-threads support.
- `.png` format output.

### Running
```bash
cargo run output.png 100x100 -1.0,0.5 0.0,-0.5
```
